#Excursion Landing Page

A sleek and responsive landing page for the fictional **Excursion** travel app, built using HTML and CSS.

##Features
- Embedded video inside a phone frame
- Clean, centered layout
- Custom styling based on design specs (font, color, spacing)
- "Coming Soon" message with download CTA

##Tech Stack
- HTML5
- CSS3
